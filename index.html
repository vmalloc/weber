Weber

Let your webapps hatch, with all the boilerplate you'll need

Takes Care of the Dirty Work

Weber provides sane defaults, utilities and methodologies to get you shipping as quickly as possible.

Ready to Deploy

Weber supports out-of-the-box deployment to servers, as well as Vagrant virtual machines. All deployment is done using Ansible, giving you powerful control over the entire process

Sane Workflows and Defaults

We spent hours tweaking our deployment, testing and development workflows so that you won't have to.

Robust

Weber is built on industry-standard, proven tools and schemes -- Flask, SQLAlchemy, Redis, Celery, Ember and more

Choose your Weber

There are three flavors of Weber, in three different levels of complexity. You get to choose which one fits your needs

Minimal

Supports the essentials: deployment, debugging, configuration, default error pages, etc. Does not contain database interfacing code.


Source Code

Backend

Like the minimal version, but also contains ORM with SQLAlchemy, migrations with Alembic, background tasks with Celery, Caching with Redis and also user/login management with Flask-Security.

Source Code

Frontend

Like the backend version, but also includes frontend support with Ember and EmberCLI.
